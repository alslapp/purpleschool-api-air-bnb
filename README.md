## Description

Домашнее задание Purpleschool: NestJS - с нуля, современный backend на TypeScript и Node JS

Простой сервис бронироновая номеров.
### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# e2e tests
$ npm run test:e2e

# unit tests
$ npm run test
```

## Дабавленные пакеты

### Nestjs packages
```bash
$ npm i @nestjs/mongoose mongoose
$ npm i class-validator class-transformer
$ npm i @nestjs/mapped-types
```

### Passport JWT
```bash
$ npm i @nestjs/jwt @nestjs/passport passport passport-jwt
$ npm i -D @types/passport-jwt
```

### node-argon2
https://www.npmjs.com/package/argon2

It's possible to hash using either Argon2i, Argon2d or Argon2id (default), and verify if a password matches a hash.

```bash
$ npm i argon2
```
